done <type task number> [eg. done 1]
-> Marks task on the list as completed.

delete <type task number> [eg. delete 2]
-> Deletes the task found on the task list.

find <type keyword> [eg. find assignment]
-> Find the tasks in task list associated with the given keyword.

date <type a date with format yyyy-mm-dd> [eg. 2021-10-01]
-> Find the tasks associated with the given date from user.

list [eg. list]
-> Lists out all tasks on the list.

bye [eg. bye]
-> Exits program
